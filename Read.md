# H5Project

本项目包含多个前端子项目，主要用于移动端和 Web 端的开发。

## 目录结构

- `OTC-register/vue-h5`：基于 Vue 的 H5 移动端项目，适用于多种业务场景。
- `treedemo`：树形结构相关的演示项目，包含多种页面和组件示例。

## 快速开始

1. 克隆仓库：
   ```sh
   git clone git@github.com:landiSky/H5Project.git
   ```
2. 进入对应子项目目录，安装依赖并运行：
   ```sh
   cd OTC-register/vue-h5
   npm install
   npm run serve
   ```
   或
   ```sh
   cd treedemo
   npm install
   npm run serve
   ```

## 主要技术栈
- Vue.js
- JavaScript/TypeScript
- uni-app（部分子项目）
